## قواعد التسمية

### أسماء الأزرار

```
✅ أضف كتاباً (فعل + اسم)
✅ احذف العنصر (فعل + اسم)
✅ احفظ التغييرات (فعل + مفعول به)
✅ تابع للتسجيل (فعل + تمييز)
❌ إضافة (اسم فقط)
❌ حذف (فعل مجرد)
❌ حفظ (فعل مجرد)
```

### أسماء الحقول

```
✅ اسم المستخدم
✅ كلمة المرور
✅ البريد الإلكتروني
✅ رقم الهاتف
✅ العنوان الكامل
❌ اسم
❌ بريد
❌ تليفون
```

### العناوين

```
✅ أضف كتاباً جديداً
✅ قائمة أمنياتك
✅ اقتراحات للقراءة
❌ إضافة (مبهمة)
❌ الأمنيات (غير واضحة)
❌ القراءة (ناقصة)
```

## الأخطاء الشائعة

### 1. استخدام الأسماء بدلاً من الأفعال

```
❌ حذف
✅ احذف العنصر

❌ حفظ
✅ احفظ التغييرات

❌ إرسال
✅ أرسل النموذج
```

### 2. استخدام رموز ترقيم زائدة

```
❌ قراءة المزيد..
✅ قراءة المزيد.

❌ البحث عن كتب ///
✅ البحث عن كتب

❌ اسم المستخدم:
✅ اسم المستخدم
```

### 3. النصوص المختصررة جداً

```
❌ ✅
✅ تم

❌ إلغاء؟
✅ هل تريد الإلغاء؟

❌ خطأ
✅ حدث خطأ
```

### 4. لغة غير مباشرة

```
❌ يجب إدخال الاسم
✅ أدخل اسمك

❌ ينبغي التحقق
✅ تحقق من البيانات

❌ لا بد من التفعيل
✅ فعّل الحساب
```

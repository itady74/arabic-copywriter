# 📝 دليل أسلوب كتابة المحتوى العربي

## إرشادات شاملة لكتابة تجربة المستخدم والمحتوى الرقمي بالعربية

هذا الدليل هو مرجع شامل لفرق العمل لتطوير محتوى عربي احترافي في التطبيقات والمواقع الإلكترونية. يستخدم عند كتابة أي نص يتفاعل معه المستخدم.

---

## القسم الأول: المبادئ الأساسية

### 1.1 لماذا نحتاج دليل أسلوب؟

| السبب       | الوصف                        |
| ----------- | ---------------------------- |
| **الاتساق** | نفس الصوت والنبرة في كل مكان |
| **الوضوح**  | محتوى يفهمه الجميع           |
| **التجربة** | تحويل المستخدم من نقطة لأخرى |
| **الثقة**   | محتوى يزيل الشك ويزيد اليقين |

### 1.2 الصوت والنبرة

**الصوت** = شخصية العلامة التجارية (ثابت دائماً)
**النبرة** = تتغير حسب الموقف والسياق

#### المبادئ الثلاثة للصوت

| المبدأ     | افعل         | لا تفعل         |
| ---------- | ------------ | --------------- |
| **الحكمة** | إيجابي وهادئ | سلبي أو اتهمامي |
| **الفضول** | اسأل واهتم   | تفرض أو أمر     |
| **الثقة**  | واضح وصريح   | غامض أو ملتبس   |

#### أربعة أبعاد للنبرة

```
     مرحة ←————————————→ جادة
     |
     |
  رسمية ←————————————→ غير رسمية
     |
     |
محترمة ←————————————→ غير موقرة
     |
     |
حماسية ←————————————→ واقعية
```

### 1.3 ملاءمة السياق العاطفي

#### ✅ المواقف الإيجابية (تشجيع، نجاح، ترحيب)

| افعل                | لا تفعل               |
| ------------------- | --------------------- |
| كن مشجعًا وإيجابيًا | لا تنسب الفضل لنفسك   |
| عبر عن اهتمامك      | لا تفترض أن الأمر سهل |
| هنّأ المستخدم       | لا تكن متعالياً       |

**أمثلة:**

```
✅ تم إرسال اقتراحك بنجاح! شكراً لمشاركتك القيمة.
❌ تم بنجاح.
```

```
✅أحسنت! أضفت الكتاب إلى قائمتك المفضلة.
❌ تمت الإضافة.
```

#### ⚪ المواقف المحايدة (معلومات، تعليمات)

| افعل                 | لا تفعل                       |
| -------------------- | ----------------------------- |
| عبر عن الثقة         | لا تولّد الشك                 |
| اربط الإجراء بالقيمة | لا تملي على المستخدم          |
| كن مباشراً           | لا تكن فضولياً أكثر من اللازم |

**أمثلة:**

```
✅ أدخل بريدك الإلكتروني للمتابعة
❌ يرجى إدخال البريد الإلكتروني أولاً
```

```
✅ أضف الكتاب للمفضلة للحصول على اقتراحات أفضل
❌ يمكنك إضافة الكتاب للمفضلة إذا أردت
```

#### ❌ المواقف السلبية (أخطاء، فشل، إلغاء)

| افعل                | لا تفعل                |
| ------------------- | ---------------------- |
| كن صادقاً ومباشراً  | لا تقلل من المشكلة     |
| وضح الخطوات التالية | لا تستخدم كلمات سلبية  |
| استخدم لغة حوارية   | لا تستخدم مصطلحات فنية |
| اعتذر بوضوح         | لا تستخدم رموز تعبيرية |

**أمثلة:**

```
✅ عذراً، تعذر إرسال اقتراحك. يرجى المحاولة مرة أخرى.
❌ فشل في الإرسال.
```

```
✅ تأكد من صحة البريد الإلكتروني وأعد المحاولة
❌ البريد الإلكتروني غير صالح
```

## القسم الثاني: طرق الكتابة الفعّالة

### 2.1 العناوين الرئيسية والفرعية

**ثلاثة أهداف:**

1. تحديد موضع المستخدم الحالي
2. فهم تنظيم المحتوى
3. توقّع الإجراءات المتاحة

**القواعد:**

- ✅ إيجاز - كلمات قليلة ومعنى كبير
- ✅ جمل مكتملة
- ❌ لا علامات ترقيم زائدة

| صواب            | خطأ              |
| --------------- | ---------------- |
| القراءة التالية | القراءة التالية. |
| كتبك المفضلة    | المفضلة:         |
| طلب جديد        | طلب جديد >>>     |

### 2.2 النصوص المصغّرة (Microcopy)

**القاعدة الذهبية:** احذف كل ما يمكن حذفه

| صواب       | خطأ              |
| ---------- | ---------------- |
| إضافة كتاب | إضافة هذا الكتاب |
| حذف        | حذف هذا العنصر   |
| حفظ        | حفظ التغييرات    |

### 2.3 الجُمل

#### كتابة الإجراء قبل النتيجة

| صواب                    | خطأ                         |
| ----------------------- | --------------------------- |
| أضف الكتاب للمفضلة      | يظهر في المفضلة عند الإضافة |
| سجّل الآن وابدأ القراءة | ابدأ القراءة بعد التسجيل    |

#### استخدام الأفعال المباشرة

| صواب          | خطأ              |
| ------------- | ---------------- |
| سنرد المبلغ   | سنقوم برد المبلغ |
| سنعلمك        | سنقوم بإعلامك    |
| سيرسل الإشعار | سيتم إرسال إشعار |

#### تجنب اللغة الجوايز

| صواب                            | خطأ                             |
| ------------------------------- | ------------------------------- |
| حدّث قائمتك للحصول على اقتراحات | حدّث قائمتك واستمتع بالاقتراحات |
| سجّل للحصول على ميزات حصرية     | سجّل واستمتع بميزات حصرية       |

## القسم الثالث: عناصر الواجهة

### 3.1 الأزرار

#### أزرار الإجراءات (فعل + اسم)

| صواب          | خطأ   |
| ------------- | ----- |
| أضف كتاباً    | إضافة |
| ابحث عن كتاب  | بحث   |
| أرسل الاقتراح | إرسال |

#### أزرار التأكيد (فعل + نتيجة)

| صواب                             | خطأ                        |
| -------------------------------- | -------------------------- |
| هل تريد الحذف؟ إلغاء / حذف       | هل تريد الحذف؟ نعم / لا    |
| هل تريد المتابعة؟ متابعة / إلغاء | هل تريد المتابعة؟ نعم / لا |

#### أزرار التحول (قيمة + صلة)

| صواب                   | خطأ         |
| ---------------------- | ----------- |
| أرسل هدية مجانية لصديق | أرسل هدية   |
| احصل على نسخة مجانية   | تحميل مجاني |
| شغّل فترة تجريبية      | تجريبي      |

#### حالة خاصة: إلغاء الحسابات

| صواب                    | خطأ                  |
| ----------------------- | -------------------- |
| غير مهتم / إلغاء الحساب | إلغاء / إلغاء الحساب |

### 3.2 الروابط

#### داخل الجمل

| صواب              | خطأ               |
| ----------------- | ----------------- |
| اقرأ المزيد.      | اقرأ المزيد.      |
| طالع شروط الخدمة. | طالع شروط الخدمة. |

#### خارج الجمل

| صواب           | خطأ           |
| -------------- | ------------- |
| المزيد         | المزيد..      |
| ليس لديك حساب؟ | ليس لديك حساب |

### 3.3 رسائل الخطأ

**ثلاثة أهداف:**

1.明确 المشكلة (ما حدث) 2.提供解决方案 (كيف الإصلاح)
3.تحويل السلبية لإيجابية

#### قواعد الكتابة

| صواب                                     | خطأ            |
| ---------------------------------------- | -------------- |
| ينبغي أن يتكون الرمز من 6 أرقام          | الرمز غير صالح |
| عذراً، لم يتم الحفظ. هل تريد المحاولة؟   | فشل الحفظ      |
| تأكد من الاتصال بالإنترنت وحاول مرة أخرى | خطأ في الشبكة  |

### 3.4 رسائل التأكيد

**المكونات:**

- عنوان: سؤال واضح
- محتوى: توضيح العواقب
- أزرار: إجراء واضح

| صواب                            | خطأ                        |
| ------------------------------- | -------------------------- |
| هل تريد تجاهل التغييرات؟        | تجاهل؟                     |
| لا يمكن التراجع عن هذا الإجراء. | سيتم حذف البيانات نهائياً. |
| تجاهل / إلغاء                   | نعم / لا                   |

### 3.5 رسائل النجاح

**ثلاثة أهداف:**

1. تأكيد الإجراء
2. عرض الخطوات التالية
3. منح شعور إيجابي

#### تحدث عن المستخدم، ليس الإجراء

| صواب              | خطأ               |
| ----------------- | ----------------- |
| تم إرسال اقتراحك! | تم الإرسال بنجاح. |
| مرحباً بعودتك!    | تم تسجيل الدخول.  |

#### أضف قيمة ومعنى

| صواب                                     | خطأ               |
| ---------------------------------------- | ----------------- |
| تمت الإضافة. الآن يمكن لأصدقائك القراءة. | تمت إضافة الكتاب. |
| تم حفظ قراءتك! ستظهر في ملفك الشخصي.     | تم الحفظ.         |

### 3.6 الحالات الفارغة

**ليست فشلاً - بل فرصة!**

| صواب                 | خطأ                |
| -------------------- | ------------------ |
| ستظهر كتبك هنا       | عذراً، لا توجد كتب |
| ابدأ بإضافة أول كتاب | لا توجد كتب بعد    |
| اقرأ أول اقتراح هنا  | لا توجد اقتراحات   |

### 3.7 المواضع المؤقتة (Placeholders)

#### الأسئلة (سؤال مباشر)

```
ما اسم الكتاب؟
من هو المؤلف؟
```

#### الفئات (إرشاد للاختيار)

```
تصفية حسب: المؤلف، النوع، التقييم
ترتيب حسب: الأحدث، الأكثر شعبية
```

#### الأمثلة (تبسيط الفهم)

```
مثال: "ماذا حدث للمصريين"
مثال: "تل SC-101"
```

### 3.8 رسائل المساعدة

**قاعدة:** كلما قلت كلما كان أفضل

| صواب          | خطأ                                     |
| ------------- | --------------------------------------- |
| 8-20 حرفاً    | يجب أن تكون كلمة المرور بين 8 و20 حرفاً |
| بريد@مثال.com | أدخل بريدك الإلكتروني الصحيح            |

## القسم الرابع: الأسلوب والآلية

### 4.1 الاختصارات

| صواب | خطأ |
| ---- | --- |
| سؤال | س   |
| صفحة | ص   |
| صورة | ص   |

### 4.2 المبني للمعلوم vs المبني للمجهول

| صواب                | خطأ                  |
| ------------------- | -------------------- |
| أضفت الكتاب للقائمة | أُضيف الكتاب للقائمة |
| أرسلت الرسالة       | أُرسلت الرسالة       |

#### مع التلطف

| صواب                       | خطأ                        |
| -------------------------- | -------------------------- |
| البريد الإلكتروني غير صحيح | البريد الذي أدخلته خاطئ    |
| الرقم غير معروف            | الرقم الذي أدخلته غير صحيح |

### 4.3 الكتابة الحوارية

| صواب            | خطأ           |
| --------------- | ------------- |
| اختر هذا الكتاب | اختر كتاب     |
| حفظ تغييراتك    | حفظ التغييرات |
| شاركنا رأيك     | شارك رأيك     |

### 4.4 الأرقام

| صواب       | خطأ               |
| ---------- | ----------------- |
| لديك 3 كتب | لديك ثلاثة كتب    |
| خصم 20%    | خصم عشرين بالمائة |
| منذ يومين  | منذ يومان         |

#### الأرقام الإنجليزية (فاصلة)

```
1,000
144
1,200,000
```

### 4.5 التواريخ

```
الخميس، أبريل ١٦
١٦ أبريل
الأمس، اليوم، غداً
الأسبوع الحالي، الأسبوع القادم
```

### 4.6 الوقت

| صواب        | خطأ         |
| ----------- | ----------- |
| 5 مساء      | 5 م         |
| 8:30 صباحاً | 8:30 ص      |
| 06:30–18:00 | 6:30 - 6:00 |

### 4.7 النقود

```
1000 جنيه مصري
250 ريال سعودي
```

### 4.8 علامات الترقيم

- **:** النقطتان - لعرض قائمة
- **،** الفاصلة - لفصل جملتين مرتبطتين
- **–** الشرطة القصيرة - لنطاق (10–20)
- **—** الشرطة الطويلة - لفصل جمل (هذا — وهو)
- **...** القطع الناقص - لحذف جزء

## القسم الخامس: إمكانية الوصول

### 5.1 مبادئ Accessibility

1. **الوضوح**: لغة يفهمها الجميع
2. **الإيجاز**: كلما كان أقصر كان أفضل
3. **التسلسل**: من الأهم للأقل أهمية
4. **الاتساق**: نفس المصطلحات

### 5.2 كتابة للأشخاص ذوي الإعاقة

| نستخدم      | لا نستخدم   |
| ----------- | ----------- |
| أصحاب الهمم | المعاقون    |
| ذوي الإعاقة | أشخاص عادون |
| يتعايشون مع | مصاب بـ     |
| يحتاج دعم   | غير قادر    |

### 5.3 النص البديل للصور

```
✅ صورة: غلاف كتاب "ملك适用的"
❌ صورة: image.jpg
```

## القسم السادس: قوائم الكلمات

### 6.1 المصطلحات المفضلة

| بدلاً من | استخدم           |
| -------- | ---------------- |
| تشغيل    | تسلسل / تشغيل    |
| قبول     | موافق            |
| إلغاء    | إغلاق / خروج     |
| رجوع     | عودة / السابق    |
| تحديد    | اختيار           |
| إدارة    | تحرير            |
| تحويل    | تبديل            |
| رؤية     | عرض              |
| يجب      | يحتاج / يجب عليك |
| تنزيل    | تصدير / تحميل    |
| إحالة    | مشاركة           |

### 6.2 أفعال الأزرار

| الإجراء | النص العربي     |
| ------- | --------------- |
| إضافة   | أضف، أنشئ، جديد |
| حذف     | احذف، أزل       |
| تعديل   | حرّر، عدّل      |
| حفظ     | احفظ، تم        |
| إرسال   | أرسل،ابعث       |
| تسجيل   | سجّل، ادخل      |
| مشاركة  | شارك، أرسل      |

## القسم السابع: أمثلة عملية

### 7.1 نموذج تسجيل الدخول

```
مرحباً بعودتك
البريد الإلكتروني
كلمة المرور
تسجيل الدخول
نسيت كلمة المرور؟
ليس لديك حساب؟ سجل الآن
```

### 7.2 نموذج إضافة منتج

```
اسم المنتج
الوصف (اختياري)
السعر
أضف للمنتجات
```

### 7.3 رسالة ترحيب

```
مرحباً بك في [اسم التطبيق]! 🎉
نحن سعيدون بانضمامك.
استكشف الميزات الجديدة:
• اكتشف كتباً جديدة
• أنشئ قائمتك المفضلة
• شارك اقتراحاتك
ابدأ الآن →
```

### 7.4 رسالة خطأ في الدفع

```
عذراً، تعذر معالجة دفعتك.
السبب: الرفض من البنك
ماذا يمكنك fazer:
• تحقق من بيانات البطاقة
• جرب طريقة دفع أخرى
• تواصل معibank
أعد المحاولة →
```

### 7.5 رسالة نجاح الاشتراك

```
✅ تم الاشتراك بنجاح!
شكراً لانضمامك.
ماذا الآن:
• استكشف الخطط المتاحة
• أضف طريقة الدفع
• راجع الفواتير
العودة للرئيسية →
```

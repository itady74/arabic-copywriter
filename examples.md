## أمثلة رسائل الخطأ

### خطأ في نموذج التسجيل

```
يرجى تصحيح الأخطاء التالية:

البريد الإلكتروني
❌ يرجى إدخال بريد إلكتروني صحيح

كلمة المرور
❌ يجب أن تكون كلمة المرور 8 أحرف على الأقل

[إصلاح وحفظ]
```

### خطأ في الدفع

```
عذراً، تعذر إتمام الدفع

المشكلة: تم رفض البطاقة
السبب: Funds insufficient

ماذا يمكنك fazer:
✓ تحقق من رصيد بطاقتك
✓ جرب بطاقة أخرى
✓ استخدم طريقة دفع مختلفة

[حاول مرة أخرى]
[طرق دفع أخرى]
```

### خطأ في الشبكة

```
عذراً، انقطع الاتصال

تحقق من اتصالك بالإنترنت وحاول مرة أخرى.

[إعادة المحاولة]
```

### خطأ 404

```
الصفحة غير موجودة

عذراً، الصفحة التي تبحث عنها غير موجودة أو تم نقلها.

[العودة للرئيسية]
[تواصل معنا]
```

## أمثلة التأكيد

### تأكيد الحذف

```
هل أنت متأكد؟

سيتم حذف "[اسم العنصر]" نهائياً ولا يمكن التراجع عن هذا الإجراء.

[إلغاء] [حذف]
```

### تأكيد الإلغاء

```
هل تريد إلغاء العملية؟

التغييرات التي أجريتها لن يتم حفظها.

[العودة للتعديل] [إلغاء]
```

```

لديك تغييرات غير محفوظة. هل تريد المغادرة بدون حفظ؟

[العودة] [مغادرة]
```
